## Codebase Patterns
- .env must use Docker service names as hostnames: DB_HOST=mariadb, REDIS_HOST=redis, MAIL_HOST=mailpit (not 127.0.0.1)
- MariaDB Docker credentials: user=pemiq, password=pemiq, database=pemiq
- User model uses traditional $fillable/$hidden/$casts properties (not PHP 8.4 #[Fillable] attribute syntax)
- Run migrations via: docker compose exec app php artisan migrate
- PHP version: 8.4.22, Laravel: 13.15.0, Filament: v4.11.7
