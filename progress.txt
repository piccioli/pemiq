## Codebase Patterns
- .env must use Docker service names as hostnames: DB_HOST=mariadb, REDIS_HOST=redis, MAIL_HOST=mailpit (not 127.0.0.1)
- MariaDB Docker credentials: user=pemiq, password=pemiq, database=pemiq
- User model uses traditional $fillable/$hidden/$casts properties (not PHP 8.4 #[Fillable] attribute syntax)
- Run migrations via: docker compose exec app php artisan migrate
- PHP version: 8.4.22, Laravel: 13.15.0, Filament: v4.11.7
- Filament Color API: use `Color::hex('#rrggbb')` for custom hex colors (not `Color::fromHex()`)
- User model implements FilamentUser interface for panel access control; `canAccessPanel()` checks Spatie role
- Laravel 11: `Registered → SendEmailVerificationNotification` must be explicitly registered in AppServiceProvider::boot()
- Auth views use `resources/views/layouts/auth.blade.php` with Tailwind CDN (no compiled assets needed for auth pages)
- Authenticated app views use `resources/views/layouts/app.blade.php` (navbar + Alpine.js + flash messages)
- Named route 'login' must exist before any auth middleware is applied (redirect target for unauthenticated users)
- Alpine.js CDN: `<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js">` (defer is required)
- Flash messages: app layout checks session('status'), session('success'), session('error') and $errors
