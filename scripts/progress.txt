## Codebase Patterns
- Tailwind v4 uses `@theme inline { --color-X: var(--X); }` in app.css — no tailwind.config.js
- Vite build via `@tailwindcss/vite` plugin; entry: resources/css/app.css + resources/js/app.js
- Blade layouts: resources/views/layouts/app.blade.php (authenticated), auth.blade.php (auth pages)
- Alpine.js and ApexCharts loaded via resources/js/app.js
- Lucide Icons loaded via CDN script in app.blade.php; call lucide.createIcons() on DOMContentLoaded AND livewire:navigated
- DS token brand color: #16D4B4 (teal-500); secondary chart color: #4F8DF5 (zone-2)
- Zone sport mapping: Run=zone3 (green), Ride=zone2 (blue), Swim=zone1 (gray)
- `@if/@endif` directives CANNOT be used inside Blade component tag attribute areas (`<x-card @if(...) attr @endif>` fails) — use an inner element with the conditional attribute instead (e.g. `<span wire:poll.5000ms hidden>` inside the card slot)
