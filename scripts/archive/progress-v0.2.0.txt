## Codebase Patterns
- Tests use SQLite :memory: — enum values in original migrations must include all valid statuses (SQLite enforces CHECK constraints from enums); for MySQL-only changes use DB::getDriverName() check in migrations
- Http::fake() stubs are permanent for the test duration (not single-use unless using Http::sequence())
- Job exceptions propagate synchronously when QUEUE_CONNECTION=sync — use try/catch in tests, not $this->expectException(), when you need DB assertions after the throw
- To avoid sleep() in tests when testing Http 429 exhaustion, mock StravaApiService directly to throw StravaRateLimitException
- StravaAccount tokens are stored encrypted; compare $account->access_token (decrypts automatically via Eloquent cast)
- Spatie permission roles must be created in setUp() even for tests that don't use roles (prevents cache issues)
